# Classificador de nível de Herói

Script em JavaScript que lê o `nome` e o `XP` do herói pelo terminal e imprime o nível correspondente.

Como executar:

1. Abra um terminal no diretório do projeto (ex: `d:\Projetos\Desafio-DIO-Classificador-nivel-Her-i`).
2. Execute:

```powershell
node desafio.js
```

Exemplo de uso (entrada e saída esperada):

- Entrada:
  - `Nome do herói: Arthur`
  - `XP do herói: 6500`
- Saída:
  - `O Herói de nome Arthur está no nível de Diamante`

Observações:
- O script aceita números com `.` ou `,` como separador de milhares (ex: `6.500` ou `6,500`).
